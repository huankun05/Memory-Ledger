# 拾忆·物语 — 个人物品资产管理系统 Plan

## 一、项目基础信息

1. **参赛赛道**：生活娱乐赛道
2. **产品形态**：React Native + Expo 移动端原生 App
3. **核心定位**：个人实物一体化管理工具，融合**物品位置记忆溯源 + 资产价值动态评估 + 质保权益管理 + 轻量消费记账**
4. **产品名称**：拾忆 · 物语（SUB.TAG · Memory Ledger）
   - 副标题：个人生活资产账本
   - Slogan：记录每一件物品的故事，让生活更有条理
5. **参赛周期**：
   - 报名阶段：6.16 - 7.15（✅ 已通过）
   - 初赛阶段：6.16 - 7.15（开发可交互 Demo）
   - 结果公示：7.21 - 7.23

## 二、技术架构

### 2.1 技术栈

| 层面 | 选型 | 版本 |
|---|---|---|
| **框架** | React Native + Expo | 0.85 / SDK 56 |
| **语言** | TypeScript | - |
| **状态管理** | Zustand | - |
| **导航** | React Navigation v7 | - |
| **数据库** | Expo SQLite | - |
| **主题** | Custom ThemeContext | - |
| **国际化** | i18next | - |
| **图标** | Lucide React Native | - |
| **图像识别** | PaddleOCR (ONNX) + ML Kit | - |
| **背景移除** | U2Net (ONNX) | - |

### 2.2 项目结构


## 四、设计系统

### 4.1 色彩系统

| 层级 | 浅色 | 深色 |
|---|---|---|
| 背景 | #FCF9F8 | #0D0D0D |
| Surface | #F5F1E8 | #1A1A1A |

| 第三色 | #396758 | #BCECDA |
| 错误 | #BA1A1A | #FFB4AB |

### 4.2 设计语言

- **核心理念**：博物馆档案室 × 生活杂志 × 液态玻璃
- **质感**：毛玻璃、大圆角、多层叠加、内发光
- **适配**：所有页面使用 `useSafeAreaInsets()` 和 `useWindowDimensions()`
