# 产品介绍书：拾忆 · 物语（SUB.TAG · Memory Ledger）

> 个人全生命周期物品记忆溯源与价值资产管理系统

---

## 一、产品概述

### 1.1 产品定位
**「拾忆 · 物语」** 是一款面向个人用户的物品全生命周期管理应用，旨在帮助用户系统化管理个人实物资产，解决「找不到东西」「忘记质保到期」「闲置物品堆积」三大核心痛点。

### 1.2 核心价值主张
- **不再丢失**：每一次物品移动都有记录，轨迹可追溯
- **不再浪费**：实时资产价值评估，折旧一目了然
- **不再错过**：质保到期智能提醒，权益不再流失

### 1.3 产品形态
- **移动端 Web App（PWA）**：支持 iOS / Android / 桌面浏览器
- **离线优先**：数据本地存储，随时随地可用
- **AI 驱动**：智能拍照识别，一键建档

## 二、目标用户

### 2.1 核心用户画像
| 用户群体 | 特征 | 痛点 |
|---|---|---|
| **都市白领** | 25-35岁，居住空间有限，数码产品多 | 物品堆积、找不到充电器、忘记保修 |
| **品质生活家** | 注重生活品质，有一定收藏爱好 | 藏品管理、保值评估、真伪溯源 |
| **家庭管理者** | 负责家庭资产采购与维护 | 家电质保管理、物品位置规划 |
| **极简主义者** | 追求断舍离，关注物品利用率 | 闲置物品识别、处置建议 |

### 2.2 用户场景
- **日常收纳**：收拾房间时记录物品位置
- **购物后**：开箱拍照，自动建档

## 四、技术特色

### 4.1 AI 能力
- **图像识别**：调用豆包视觉大模型，识别准确率 96%+
- **自动分类**：智能判断物品品类，减少手动输入
- **价格预估**：基于识别结果估算购入价格
- **降级策略**：API 不可用时自动切换到本地物品库匹配

### 4.2 资产估值算法
采用**复利递减法**计算物品残值：

```
残值 = 购入价 × (1 - 折旧率)^使用年限
```

| 品类 | 年折旧率 | 说明 |
|---|---|---|
| 数码产品 | 25% | 更新换代快 |
| 家用电器 | 18% | 寿命较长 |
| 衣物 | 35% | 季节性强 |
| 书籍 | 10% | 折旧慢 |
| 收藏品 | 5% | 可能升值 |

### 4.3 主题系统
- **浅色模式**：纸墨书香，温暖柔和（默认）
- **深色模式**：暗夜静谧，护眼舒适
- **跟随系统**：自动匹配系统设置
- **平滑过渡**：0.3秒动画切换

### 4.4 数据存储
- **前端**：IndexedDB（Dexie.js）本地存储，离线可用
- **后端**：SQLite 数据库，支持云同步（未来扩展）
- **隐私保护**：数据不上传第三方，完全自主掌控

## 六、使用流程

### 6.1 新用户上手
1. **打开应用** → 自动初始化示例数据
2. **浏览首页** → 查看资产总览
3. **AI 建档** → 拍照识别物品
4. **完善信息** → 补充价格/位置/质保
5. **管理位置** → 设置房间结构

### 6.2 日常使用
1. **找东西** → 搜索或查看位置地图
2. **移动物品** → 更新位置，记录轨迹
3. **查看折旧** → 在详情页查看贬值曲线
4. **质保提醒** → 首页收到到期通知
5. **导出备份** → 定期导出数据
